## • Configuration Essentielle

Avant de profiter de la puissance d'**Absolute Translate**, il est vital de configurer votre environnement linguistique. Vous pouvez accéder rapidement aux paramètres en effectuant un **triple appui** sur la touche `Alt + Windows + T` ou en naviguant via le Menu NVDA > Paramètres > Paramètres > **Absolute Translate**.

* **Paire de Langues Principale:** Réglez la langue source sur "Auto" pour une détection sans effort et choisissez votre langue cible préférée.
* **Le Système d'Inversion (Critique):** Définissez ici votre langue secondaire. Lorsque vous devez traduire à nouveau vers la source d'origine, l'extension utilise ce paramètre pour inverser instantanément les directions.
* **Inversion Automatique:** Nous recommandons vivement de cocher cette option. Elle permet au moteur d'inverser intelligemment les directions de traduction en fonction du texte détecté, sans intervention manuelle.
* **Gestion des Réponses:** Sélectionnez "Copier dans le presse-papiers" pour sauvegarder vos résultats. Si vous activez le mode **"Ajouter"**, les nouvelles traductions seront ajoutées à la fin du contenu existant au lieu de l'écraser.
* **Mode Continu:** Laissez cette option cochée pour gérer les contenus longs via notre interface spécialisée.

## • Description

Éliminez les frictions liées au changement d'application pour comprendre le monde. **Absolute Translate** est un pont haute performance entre vous et l'information globale. Intégré directement dans votre lecteur d'écran, il fournit des traductions ultra-rapides du texte à l'écran, vous permettant de rester concentré et productif dans n'importe quelle langue.

<br>

## • Utilisation (La logique multi-appuis)

L'extension utilise un système sophistiqué de contrôle par touche unique via `Alt + Windows + T`. Le comportement s'adapte à vos besoins selon le nombre d'appuis :

1.  **Simple appui:** Traduit le **texte sélectionné** (surligné) actuellement sur votre écran.
2.  **Double appui:** Traduit le **dernier texte prononcé** par NVDA. Idéal pour les notifications système, les messages d'erreur ou les dialogues éphémères qui ne peuvent pas être surlignés.
3.  **Triple appui:** Votre raccourci ultime vers la **boîte de dialogue des paramètres**. Ajustez vos langues ou vos préférences instantanément sans parcourir les menus.

<br>

## • Fonctionnalités Avancées

### Gestion de Contenu à Haut Volume
Pour les documents étendus ou les articles longs, l'extension déclenche automatiquement le **mode traduction longue** dès que le texte dépasse **2 000 caractères** :

* **Segmentation Intelligente en Blocs:** Pour maintenir une qualité de traduction optimale, le texte est divisé en blocs gérables allant jusqu'à **5 000 caractères**.
* **Traduction Séquentielle:** Utilisez le bouton **"Continuer"** (ou `Alt+C`) pour traiter et lire le bloc de texte suivant dans la séquence.
* **Intégration Intelligente du Presse-papiers:** Si 'Copier dans le presse-papiers' est actif, chaque bloc est sauvegardé. Avec **'Ajouter'** activé, le deuxième bloc est joint de manière fluide au premier, vous permettant de capturer un article entier en une seule fois.
* **Inversion de Vue Double (Alt+S):** Basculez entre la 'Source Originale' et le 'Résultat Traduit' au sein de la boîte de dialogue pour comparer le contexte.

<br>

> **Points Forts:** La véritable efficacité va au-delà de la vitesse ; il s'agit d'une compréhension totale. Absolute Translate garantit que chaque information sur votre écran est accessible, quelle que soit la langue dans laquelle elle a été écrite.
