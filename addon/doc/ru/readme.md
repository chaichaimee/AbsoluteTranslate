## • Основные настройки

Прежде чем использовать возможности **Absolute Translate**, необходимо настроить языковую среду. Вы можете быстро перейти к настройкам, **трижды нажав** горячую клавишу `Alt + Windows + T`, или через меню NVDA > Параметры > Настройки > **Absolute Translate**.

* **Основная языковая пара:** Установите исходный язык на "Авто" для автоматического определения и выберите желаемый целевой язык.
* **Система переключения (Критически важно):** Укажите здесь ваш дополнительный язык. Когда вам потребуется перевести обратно на исходный язык, дополнение использует эту настройку для мгновенной смены направления.
* **Автопереключение:** Мы настоятельно рекомендуем включить эту опцию. Она позволяет движку интеллектуально менять направление перевода в зависимости от обнаруженного текста без вмешательства пользователя.
* **Обработка ответов:** Выберите "Копировать в буфер обмена", чтобы сохранить результаты. Если включен режим **"Добавление"**, новые переводы будут добавляться в конец содержимого буфера, а не перезаписывать его.
* **Непрерывный режим:** Оставьте этот флажок включенным для работы с объемными текстами через специальный интерфейс.

## • Описание

Забудьте о необходимости переключаться между приложениями, чтобы понять мир. **Absolute Translate** — это высокопроизводительный мост между вами и мировой информацией. Интегрируясь непосредственно в вашу программу чтения с экрана, он обеспечивает молниеносный перевод текста, позволяя вам оставаться сосредоточенным и продуктивным на любом языке.

<br>

## • Как использовать (Логика нескольких нажатий)

Дополнение использует сложную систему управления одной клавишей через `Alt + Windows + T`. Поведение меняется в зависимости от количества нажатий:

1.  **Одиночное нажатие:** Переводит **выделенный текст** (подсвеченный), который сейчас на экране.
2.  **Двойное нажатие:** Переводит **последний текст**, произнесенный NVDA. Идеально для системных уведомлений, сообщений об ошибках или временных окон, которые нельзя выделить.
3.  **Тройное нажатие:** Ваш быстрый доступ к **диалогу настроек**. Мгновенно меняйте языки или предпочтения, не блуждая по меню.

<br>

## • Расширенные возможности

### Управление большими объемами текста
Для объемных документов или длинных статей дополнение автоматически включает **режим длинного перевода**, если текст превышает **2000 символов**:

* **Интеллектуальное деление на блоки:** Для поддержания высокого качества перевода текст делится на управляемые блоки объемом до **5000 символов**.
* **Последовательный перевод:** Используйте кнопку **"Продолжить"** (или `Alt+C`) для обработки и чтения следующего блока текста.
* **Интеллектуальная интеграция с буфером обмена:** Если включено копирование в буфер, каждый блок сохраняется. В режиме **"Добавление"** второй блок плавно присоединяется к первому, позволяя сохранить всю статью целиком за один раз.
* **Смена вида (Alt+S):** Переключайтесь между оригиналом и переводом прямо в диалоговом окне для сверки контекста.
